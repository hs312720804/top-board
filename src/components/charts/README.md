# 各类图表展示使用说明
## 参数说明

 参数  | 说明  | 类型  |  可选值  |  默认值
 ---- | ----- | ------ | ------ | ------  
 type  | 图表类型 | String | bar/line/pie/funnel/scatter | bar
 id  | 图表ID | String | -- | --
 option  | 展示数据 | Object | 参考echarts官网 | --
 theme  | 主题 | String | black/white | white

### type说明

 参数  | line  | bar  | pie  | funnel  | scatter  | bullet | heatmap
 ---- | ----- | ----- | ----- | ----- | ----- | ----- | -----
 说明 | 拆线图 | 柱状图 | 饼图 | 漏斗图 | 散点图 | 子弹图 | 热力图
